## 💻 Getting Started <a name="getting-started"></a>

- To get a local copy up and running, follow these steps.

### Prerequisites

- A code editor of your choice (like vs code or atom and so on)
-version control system (git is preferred)

### Setup

- Create a new folder to store the files in.

### Install

Install this project with:

- Open your command prompt or terminal and run.
- git clone https://github.com/Kanyorok/Capstone-Grp_Kanban.git
- cd Capstone-Grp_Kanban
- npm i
- npm start (To Launch App).


### Usage

- this is a Website Application app and I will use it to insert my upcoming projects.

### Run tests

To run tests, run the following command:

- npm test
- npm lint
- npm start

### Deployment

You can deploy this project using:

```
  github pages
```
